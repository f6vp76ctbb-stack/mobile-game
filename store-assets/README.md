# Store-Assets (Play Console)

Fertige Grafiken zum Hochladen im Play-Console-Store-Eintrag. Erzeugt aus dem
echten Web-Build (Aurora-Theme). Captions siehe `docs/STORE-LISTING.md`.

| Datei | Verwendung | Format |
|---|---|---|
| `app-icon-512.png` | App-Symbol (deckend, keine Transparenz) | 512×512 |
| `feature-graphic-1024x500.png` | Feature-/Vorstellungsgrafik (Kopfbanner) | 1024×500 |
| `screenshot-1-home.png` | Screenshot: Startbildschirm / „Komplett offline spielbar" | 1080×1920 |
| `screenshot-2-gameplay.png` | Screenshot: Spielbrett / „Räume Reihen & Spalten" | 1080×1920 |
| `screenshot-3-themes.png` | Screenshot: Themes / „Themes zum Freischalten" | 1080×1920 |
| `screenshot-4-missions.png` | Screenshot: Missionen / „Missionen & Münzen" | 1080×1920 |

Hinweise:
- **Telefon-Screenshots** (min. 4): die vier `screenshot-*.png` (1080×1920, 9:16).
- **Tablet-Screenshots (7" + 10")**: dieselben vier Screenshots erfüllen die Vorgaben
  (9:16, 1080 px) und können in beide Tablet-Felder hochgeladen werden.
- **Optional/überspringen:** Video, Google Play Games auf PC, Chromebook, Android XR.
- Für den Store reichen 2 Screenshots; erlaubt sind bis zu 8. Reihenfolge oben ist ein Vorschlag.
- Der angezeigte Bestwert (12 840) und der Name „Puzzlerin" sind reine Demo-Werte für die Aufnahme.
- Optional nachlegbar: Combo-Fieber (glühendes Board), Bestenliste, Tägliche Challenge.
